Burguer King

Está destinada a un público más casual, la información viene clara y concisa, acompañado de elementos visuales que ayudan a entender los tipos de productos y menus

Usa una paleta de color asociada a la marca y también con colores que psicológicamente están asociados al hambre como el rojo, amarillo y naranja

Los elementos están bien distribuidos gracias a los espacios en blanco. Tiene algunos elementos animados simples para mostrar diferencias entre hamburguersas pero no resulta cargante para el usuario

Es una web responsive y preparada para móviles ya que mucho de su público directamente mira las ofertas de la web desde el móvil

Restaurante Baobab

Sin embargo algunas transiciones molestan visualmente como las frases centrales
