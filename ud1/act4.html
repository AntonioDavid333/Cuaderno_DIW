ACTIVIDAD 4

Burguer King

Está destinada a un público más casual, la información viene clara y concisa, acompañado de elementos visuales que ayudan a entender los tipos de productos y menus

Usa una paleta de color asociada a la marca y también con colores que psicológicamente están asociados al hambre como el rojo, amarillo y naranja

Los elementos están bien distribuidos gracias a los espacios en blanco. Tiene algunos elementos animados simples para mostrar diferencias entre hamburguersas pero no resulta cargante para el usuario

Es una web responsive y preparada para móviles ya que mucho de su público directamente mira las ofertas de la web desde el móvil

Restaurante Baobab

Es una web destinada a un público más adulto y vegetariano

A priori los elementos parecen solaparse unos a otros pero conforme vas bajando la página los elementos se distribuyen y se reparten con animaciones

Tiene un estilo muy diferenciado, usa unos colores pastel para cada sección y una transición de elementos suave

Sin embargo algunas transiciones molestan visualmente como las frases centrales

Ferrari

Es una web destinada a un público muy concreto: personas amantes de los vehículos y/o de la marca

Los colores de la pagina web son elegantes: blanco, negro y gris, acompañados de detalles rojos propios de la marca. Esto se aprecia muy bien en el multimedia de la web, los videos siempre tienen estos 3 colores presentes.

Tiene una opcion de Accesibilidad para ajustar la web a distintos perfiles: a prueba de convulsiones, problemas de vision, TDAH, discapacidad cognitiva. En tiempo real la pagina de ajusta y cambia tonos, tipografías e incluso la forma de mostrar la información para que la experiencia de usuario sea la mejor posible

Volvo

Es una web diseñada para un público adulto, sobrio y con gusto por la elegancia

La página tiene un diseño sencillo y funcional que trata de erradicar todo tipo de detalle innecesario

El producto que se pretende vender son coches, por lo que las imágenes de los mismos ocupan la mayor parte de la página, así como los carteles de características

Los colores transmiten una sensacion de comodidad o confort a la vez que elegancia, pues juega con tonos beige, blanco y negros, generando una sensación de calidez a la vez que sobriedad

Cabe destacar los atractivos incluidos en la web, como la simulación de un modelo 3D de un vehículo al que se le puede cambiar el color y ver el renderizado al instante

Carolina Herrera

La página web parece diseñada para un público adulto femenino

La paleta se compone principalmente de negros y blancos, para generar contraste, y de pinceladas de color rosa, seguido de imágenes de modelos femeninas, lo que deja ver su potencial público

La interfaz es muy sencilla e intuitiva, sin demasiada información, lo que crea una experiencia cómoda para el usuario.

Además, los productos que ofrece estan muy bien ordenados y visibles, lo que evita la pérdida de tiempo o tener que buscar demasiado

Dior

La página web de Dior cumple con características similares a la web de Carolina Herrera, salvo que el potencial público, a priori, parece ser masculino

Los principales colores que observamos en la web son el blanco y el negro, salvo que en este caso, al contario de la web de Carolina Herrera, se acompaña de pinceladas de color azul junto con algún tono tierra para ofrecer estabilidad en complementación con el azul

Además, la imágen principal es de una figura masculina, lo que deja ver las intenciones de la web con respecto al público elegido

El resto de la estructura sigue las mismas características que la web anterior: sobriedad y una interfaz sencilla de manejar y de fácil navegación sin exceso de información.
